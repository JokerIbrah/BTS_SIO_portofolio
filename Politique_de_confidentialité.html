Politique de confidentialité

Dernière mise à jour : 07/11/2025


Votre vie privée est importante pour nous. Ce site ne collecte aucune donnée personnelle, ne dépose aucun cookie et n’utilise aucun service de suivi ou d’analyse (comme Google Analytics).

Données collectées

Nous ne recueillons aucune donnée nominative, adresse e-mail ou information d’identification. Les seules données éventuellement enregistrées sont celles fournies volontairement via un formulaire de contact (si applicable).

Cookies

Ce site ne place aucun cookie sur votre appareil.

Hébergement

Ce site est hébergé par GitHub Pages, un service de GitHub, Inc. GitHub peut collecter certaines données techniques (comme les journaux de serveur ou adresses IP) pour des raisons de sécurité et de maintenance. Pour plus d’informations, consultez la politique de confidentialité de GitHub : https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement
